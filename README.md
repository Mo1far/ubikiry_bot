# Cloning this repo
```
git clone https://github.com/Mo1far/ubikiry_bot.git
```
# Install requirements
```
pip3 install -r requirements.txt
```
# Running Redis
```
$ redis-server
```

# Edit TG TOKEN
insert your token in bot/config.py
```
BOT_TOKEN = 'Your Token'
```
# Run
 ```
 $ screen python3.8 main.py
 ```
